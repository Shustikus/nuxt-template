# Nuxt template

Стартовый репозиторий: Nuxt 4, TypeScript, Vitest, Storybook, MSW.

- Страницы — `app/pages/`
- Общие компоненты — `app/components/` (создайте при необходимости)
- Сторис и MDX — `app/docs/`
- Глобальные стили — `app/assets/styles/main.css`
