# Nuxt template

Стартовый репозиторий: Nuxt 4, TypeScript, Vitest, Storybook, MSW.

Инфраструктура без UI kit и бизнес-фич — только архитектурные паттерны:

- **Managed routing** — `app/router.options.ts` + `app/config/routes.ts`
- **HTTP** — `app/utils/apiFetch.ts`, `app/config/api.ts`, Nitro-прокси и `resolveRuntimeConfigString` в `server/utils/`
- **MSW** — `mocks/handlers.ts` + `app/plugins/msw.client.ts` в dev (worker из `postinstall`), `tests/setup.ts` в
  unit-тестах
- **Ошибки** — `app/error.vue` (глобальная страница с `useSeoMeta`)
- **Фичи** — добавляйте в `app/features/<name>/` (components, services, dto, mappers, …)

Структура:

- Страницы — `app/pages/`
- Компоненты — `app/components/` (создайте `ui/`, `layout/` при необходимости)
- Сторис и MDX — `app/docs/`
- Глобальные стили — `app/assets/styles/main.css`
